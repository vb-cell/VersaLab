VersaLab

VersaLab is a smart, portable physics lab using Arduino Uno. It demonstrates core physics principles using sensors and provides real-time interaction via mobile or laptop terminal.

Components & Connections

  • Arduino Uno
  • BH1750 Light Sensor (I2C) → SDA(A4), SCL(A5)
  • TCS34725 Color Sensor (I2C) → SDA(A4), SCL(A5)
  • Hall Sensor → D6
  • Red LED → D5 (via 220Ω resistor)
  • Green LED → D4 (via 220Ω resistor)
  • Power Supply: USB or 9V battery via VIN & GND

How to Conduct Experiments

Use any serial terminal app. Send a number (1–6) to choose experiment, then use these commands:

  • s – Start
  • t – Pause
  • r – Resume
  • x – Restart
  • z – Reset Values
  • m – Back to Menu
  • e – End
1. Magnetic Field Detection
  1. Connect Hall sensor to D6
  2. Send 1 then s in serial terminal
  3. Bring magnet close to sensor – LEDs blink when field detected
  4. Use e to end or m to go to menu
2. Inverse Square Law
  1. Use BH1750 at fixed point
  2. Move light source at 5, 10, 15... cm
  3. Send 2 then s to begin
  4. Lux values logged every 6 seconds
3. Color Absorption
  1. Place colored object near TCS34725 sensor
  2. Send 3 then s
  3. 8-bit RGB values displayed every 2 seconds
4. Shadow Detection
  1. Use BH1750 sensor
  2. Cover/uncover sensor while reading
  3. Send 4 then s
  4. Lux logged every 1 second
5. Color Consistency
  1. Send 5 then s
  2. If RGB change exceeds threshold → red LED blinks
6. RPM Measurement
  1. Mount magnet on wheel edge
  2. Sensor on D6 counts rotations for 10s
  3. Send 6 then s
  4. RPM calculated and displayed
