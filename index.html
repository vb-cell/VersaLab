About VersaLab

VersaLab is a versatile multi-sensor experiment station for school physics demonstrations. It allows students to perform real-time sensor-based investigations using Arduino and mobile interface.

Command Reference

  • s - Start Experiment
  • x - Stop Experiment
  • r - Restart Experiment
  • z - Reset Readings
  • m - Show Menu
  • e - End Experiment
  • 1–6 - Select Experiment

How to Conduct Experiments

1. Magnetic Field Detection
  • Connect the Hall Effect sensor to pin 4
  • Bring a magnet near the sensor
  • LEDs will blink if field is detected
  • Use commands 's' to start, 'e' to end
2. Inverse Square Law (Light)
  • Place BH1750 sensor at a point
  • Place light at 5cm, 10cm...
  • Observe lux decrease over distance
  • Use command 's' to begin step-wise logging
3. Color Absorption (RGB Sensor)
  • Use TCS34725 sensor on colored paper
  • Press 's' to get RGB readings every 2s
  • Use 'z' to zero values, 'x' to stop
4. Shadow Detection
  • Use BH1750 sensor
  • Place hand over it every second
  • Check lux dip over time
5. Color Consistency
  • Compare two papers for RGB difference
  • Red LED glows on large difference
6. RPM Measurement
  • Attach Hall sensor to rotating wheel
  • Count pulses in 10s → Calculate RPM
  • Use 's' to start, 'x' to stop, 'r' to restart

Connection Diagram

Power: Arduino 5V → Sensors VCC, GND → Sensors GND

  • Red LED – Pin 6
  • Green LED – Pin 5
  • Hall Sensor – Pin 4
  • BH1750 – I2C (A4 - SDA, A5 - SCL)
  • TCS34725 – I2C (A4 - SDA, A5 - SCL)
